Python-fitz

A set of python bindings for fitz, MuPDF's rendering library.

Download this project as a .zip file Download this project as a tar.gz file

python-fitz is a set of python bindings for MuPDF rendering library. Most of the work is done by SWIG with -builtin option on.

A simple document can be found here.

Dependencies

This module depends on MuPDF(version=1.1).

By default, fitz is compiled as a static lib. If that's the case, you need to recompile MuPDF with -fPIC option.

Install

Extract the tarball and run

python setup.py install

Run

python setup.py --help

for more information about installation.
