WriteOptions¶

Options used in Document.write_document.

Methods¶

__init__ Constructor.

Attributes¶

do_ascii Whether to make the output ascii.
do_expand Indicate Which aspect of the file should be expanded.
do_garbage Whether to garbage collect the file before writing.
do_linear Whether write linearised.

Class API¶

class WriteOptions¶
__init__(self, do_ascii=0, do_expand=fz_expand_all, do_garbage=1, do_linear=0)¶

Constructor.

do_ascii¶
If non-zero then attempt (where possible) to make the output ascii.
Type :int
do_expand¶
Bitflags; each non zero bit indicates an aspect of the file that should be ‘expanded’ on writing. See fz_expand_images, fz_expand_fonts, fz_expand_all.
Type :int
do_garbage¶
If non-zero then attempt (where possible) to garbage collect the file before writing.
Type :int
do_linear¶

If non-zero then write linearised.

Type :int
