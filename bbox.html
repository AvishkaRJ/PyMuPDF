BBox¶

BBox is a bounding box similar to Rect, except that all corner coordinates are rounded to integer coordinates.

See also

Rect

Methods¶

intersect_bbox Compute the intersection with another bounding box.
union_bbox Compute the union with another bounding box.
get_width Get width of the bounding box.
get_height Get height of the bounding box.
is_empty Check if the bounding box is empty.
is_infinite Check if the bounding box is infinite.

Attributes¶

x0 X-coordinate of the top left corner.
y0 Y-coordinate of the top left corner.
x1 X-coordinate of the bottom right corner.
y1 Y-coordinate of the bottom right corner.

Class API¶

class BBox¶
intersect_bbox(self, bbox)¶

Compute the intersection with another bounding box. Similar to Rect.intersect_rect but operates on two bounding boxes instead of two rectangles.

Parameters:bbox (BBox) – The bounding box to intersect.
Return type:BBox
union_bbox(self, bbox)¶

Compute the union with another bounding box. Similar to Rect.union_rect but operates on two bounding boxes instead of two rectangles.

Parameters:bbox (BBox) – The bounding box to union.
Return type:BBox
