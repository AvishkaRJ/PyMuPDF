Context¶

Context is an object that contains the global state. Most functions in MuPDF take a context argument to be able to reference the global state.

Use new_context to create context.

Methods¶

get_aa_level Get the number of bits of antialiasing we are using. Between 0 and 8.
set_aa_level Set the number of bits of antialiasing we should use.
