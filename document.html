Class API¶

class Document¶
load_page(self, number)¶

Load a page. After load_page is it possible to retrieve the size of the page using Page.bound_page, or to render the page using Page.run_page

Parameters:number (int) – page number, 0 is the first page of the document.
Return type:Page
load_outline(self)¶

Load the hierarchical document outline.

Return type:Outline
needs_password(self)¶

Check if a document is encrypted with a non-blank password.

Return type:int
Returns:1 if password needed, 0 otherwise.
authenticate_password(self, password)¶

Test if the given password can decrypt the document.

Parameters:password (string) – The password string to be checked.
Return type:int
Returns:1 if the password authenticated, 0 otherwise.
count_page(self)¶

Return the number of pages in document. May return 0 for documents with no pages.

Return type:int
write_document(self, filename, writeoptions)¶

Write a document out.

Save a copy of the current document in its original format. Internally the document may change.

Parameters:
  • filename (string) – The filename to save to.
